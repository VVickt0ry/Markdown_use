## 標題與文字樣式 🌈

> **核心記憶口訣**：井號越多越小，星號越疊越粗，波浪代表刪除～

### 快速演示

# 一級標題（像大標題海報）
## 二級標題（章節名常用）
### 三級標題（小節）

**粗體** vs *斜體* vs ***粗斜體*** vs ~~刪除線~~

- 快捷鍵記憶：**粗** → Ctrl+B  
- *斜* → Ctrl+I  
- ~~刪~~ → 像被劃掉的錯誤答案

<details>

__下劃線粗體__ （兩個下劃線）  
***閃爍粗斜體*** （三個星號）

## 代碼展示 💻

`單行代碼` 適合命令快速展示 → `git commit -m "fix: 修復bug"`

```python
# 多行代碼 + 趣味註釋
print("Hello Markdown！🌟")

# 記住：語言標識決定高亮顏色
# python → 綠色調
# js     → 黃橙調
# bash   → 紫色調
```



### 今天 Markdown 學習進度

- [x] 掌握標題層級
- [x] 會寫粗斜體和刪除線
- [ ] 搞懂表格對齊 :--: 和 :---:
- [ ] 學會寫腳注[^note]

[^note]: 腳注是正文不亂的小紙條～超實用！

### 無序列表小遊戲

- 🍎 蘋果代表重點
- 🔥 火代表超重要
- 🐛 bug 代表要注意的坑
